Lugi Handboll

Detta var ett ASP.NET-projekt som jag var med och utvecklade 2009. Det var ett biljettbokningssystem för handbollsklubben Lugi i Lund. Systemet användes för att hantera biljettförsäljning och bokning av biljetter till klubbens matcher.

Jag har inte tagit med någon länk till systemet eftersom det är helt omgjort idag.

Tekniker och kompetenser som jag använde i detta projektet.

HTML CSS Responsive Design JavaScript JQuery C#.Net ASP.Net SqlServer
